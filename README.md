# TWRP Samsung Galaxy A10s

## How to compile it:

To build:
export ALLOW_MISSING_DEPENDENCIES=true

setup envirnoment
`. build/envsetup.sh`

then prepare
`lunch omni_a10sxx-eng`

make the image
`mka recoveryimage`

## How to find the image built

`cd /out/target/product/a10s`

see recovery.img
